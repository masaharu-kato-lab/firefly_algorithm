# firefly_algorithm
Implementation of Firefly Algorithm

# Installation
```
python3.8 -m venv env
source env/bin/activate
pip install -r requirements.txt
```

# Running
- Before running
```
source env/bin/activate
```

```
??????.py -n 30 -g 0.01502 -a 2.0 -ba 2.0 -t 100 -d 2 -u 0 --init random --quiet --stdout

- After running
```
deactivate
